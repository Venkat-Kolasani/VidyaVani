## Visual Design Guidelines

### Color Scheme
- **Primary**: Deep Blue (#1565C0) - Trust, Education
- **Secondary**: Orange (#FF9800) - Energy, Innovation
- **Accent**: Green (#4CAF50) - Growth, Success
- **Text**: Dark Gray (#212121) - Readability
- **Background**: Light Gray (#F5F5F5) - Clean, Professional

### Typography
- **Headers**: Roboto Bold, 32-36pt
- **Subheaders**: Roboto Medium, 24-28pt
- **Body Text**: Roboto Regular, 18-20pt
- **Captions**: Roboto Light, 14-16pt

### Visual Elements
- **Icons**: Material Design icons for consistency
- **Charts**: Simple, clear data visualizations
- **Screenshots**: High-quality system interface captures
- **Diagrams**: Clean architectural representations

### Slide Layout Principles
- **Minimal Text**: Maximum 6 lines per slide
- **Visual Hierarchy**: Clear information flow
- **White Space**: Generous spacing for readability
- **Consistency**: Uniform layout across slides
